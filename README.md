
# **PCOS Women’s Health Business Intelligence Dashboard**

An **end-to-end healthcare analytics and business intelligence project** that analyzes PCOS patient data to uncover **hormonal, lifestyle, metabolic, and reproductive health patterns** using **Power BI**.

This project demonstrates how real-world women’s health data related to **Polycystic Ovary Syndrome (PCOS)** can be transformed into **actionable healthcare insights** through a complete analytics pipeline:

**Data Ingestion → Data Cleaning → Feature Engineering → KPI Creation → BI Modeling → Interactive Dashboard Visualization**

---

## **Project Overview**

This study analyzes **2,000 PCOS patient records** to identify meaningful trends and correlations across **hormonal imbalance, BMI, lifestyle habits, and reproductive indicators**. The project is designed to support **data-driven decision-making** in women’s healthcare through structured analytics and visual storytelling.

### **Workflow Includes**

* **Data ingestion and preprocessing**
* **Feature engineering** (BMI categories, hormonal averages)
* **KPI development using DAX**
* **Analytical and dimensional data modeling**
* **Interactive Power BI dashboard design**

## **Key Features**

### **1. Demographic & Patient Overview**

* **Total patient count**
* **Age distribution analysis**
* **BMI classification** (Underweight, Normal, Overweight, Obese)
* **Menstrual regularity trends**

### **2. Hormonal Health Indicators**

* **Average AMH levels**
* **Average Testosterone levels**
* **Hormonal distribution across age groups**
* **Identification of hormonal imbalance patterns in PCOS patients**

### **3. Lifestyle & Behavioral Analysis**

* **Exercise habits analysis**
* **Dietary pattern insights**
* **Sedentary vs active lifestyle comparison**
* **Lifestyle correlation with BMI and hormonal imbalance**

### **4. Reproductive & Metabolic Indicators**

* **PCOS symptom prevalence**
* **Fertility-related biomarkers**
* **BMI vs hormonal trend analysis**
* **Risk stratification using aggregated KPIs**

### **5. Interactive Power BI Dashboard**

* **Dynamic slicers** (Age, BMI, Lifestyle factors)
* **KPI cards** (Total Patients, Avg AMH, Avg Testosterone)
* **Donut charts, bar charts, and comparative visuals**
* **Clean, clinician-friendly dashboard layout**
